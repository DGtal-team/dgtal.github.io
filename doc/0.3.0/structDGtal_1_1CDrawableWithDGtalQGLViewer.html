Description of concept 'CDrawableWithDGtalQGLViewer'

Aim: The concept CDrawableWithDGtalQGLViewer specifies what are the classes that admit an export with DGtalBoard. An object x satisfying this concept may then be used as:

   DGtalBoard board;
   board << CustomStyle( x.styleName(), x.defaultStyle() )
         << x;
