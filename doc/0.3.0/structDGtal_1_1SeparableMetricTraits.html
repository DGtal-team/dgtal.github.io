template<typename TAbscissa, typename TInternalValue, DGtal::uint32_t tp>
struct DGtal::SeparableMetricTraits< TAbscissa, TInternalValue, tp >

Aim: Implements basic functions associated to metrics used by separable volumetric algorithms.

Description of template class 'SeparableMetricTraits'

Template Parameters:
TAbscissaType used to store the coordinaites of the Domain.
TInternalValuethe type used to store the internal numbers for exact computations. More precisely, TInternalValueType must be able to represent numbers of type TAbscissa to the power tp
tpthe order p of the L_p metric.
