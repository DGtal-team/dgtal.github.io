Detailed Description

Custom style class redefining the pen attributes. You may use DGtalBoard::Color::None for transparent color.

   DGtalBoard board;
   board << CustomPen( DGtalBoard::Color::Green, DGtalBoard::Color::Black,
                       3.0 );
   ...
