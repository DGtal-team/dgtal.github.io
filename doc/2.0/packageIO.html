Package Overview

In this package, we present DGtal tools and utilities to import/export images and visualize digital data using interactive (viewers) and non-interactive (boards) mechanisms.

Modules

  • Board2D: a stream mechanism for displaying 2D digital objects (Jacques-Olivier Lachaud, Nicolas Normand, David Coeurjolly, Martial Tola)
  • Display3D: a stream mechanism for displaying 3D DGtal objects (Bertrand Kerautret, Martial Tola, Aline Martin, David Coeurjolly, Bastien Doignies)
    • Migrating viewer from 1.4 to 2.0 (Bastien Doignies)
  • Image and digital object import/export (David Coeurjolly, Bertrand Kerautret, Martial Tola, Pierre Gueth)
