Detailed Description

template<typename TIterator>
class DGtal::HyperRectDomain_ReverseIterator< TIterator >

Reverse iterator for HyperRectDomain.

Template Parameters
TIteratorIterator type on HyperRectDomain
Note
We need this specific implementation of a reverse iterator instead of a std::reverse_iterator because the latter works only on non-stashing iterators (i.e. iterators that don't own its pointed data). Otherwise, it will lead to dangling reference!

doc-examples/range.cpp.
