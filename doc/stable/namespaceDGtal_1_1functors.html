Detailed Description

functors namespace gathers all DGtal functors.

Aim: Provide various lambda functions and others L-MST related functors.

Description: Extension of namespace functors by functors related to L-MST.

A lambda function \(\lambda()\) - maps from [0,1] \(\in \mathbb{R}_+\) with F(0) = F(1) = 0 and F() > 0 elsewhere and need to satisfy convexity/concavity property. For more information see J.-O. Lachaud et el [72].
