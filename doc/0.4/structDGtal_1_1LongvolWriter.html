Detailed Description

template<typename TImage, typename TColormap>
struct DGtal::LongvolWriter< TImage, TColormap >

Aim: Export a 3D Image using the Longvol formats (volumetric image with DGtal::uint64_t value type).

Description of template struct 'LongvolWriter'

The file format contains an ASCII header and a raw binary array (little-endian uint64t).

Template Parameters:
TImagethe Image type.
TColormapthe type of the colormap to use in the export.
