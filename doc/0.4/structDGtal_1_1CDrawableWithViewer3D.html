Description of concept 'CDrawableWithViewer3D'

Aim: The concept CDrawableWithViewer3D specifies what are the classes that admit an export with Board2D. An object x satisfying this concept may then be used as:

   Board2D board;
   board << CustomStyle( x.styleName(), x.defaultStyle() )
         << x;

  • X : A type that is a model of CDrawableWithViewer3D
  • x, y : Object of type X
