Description of concept 'CDrawableWithBoard3DTo2D'

Aim: The concept CDrawableWithBoard3DTo2D specifies what are the classes that admit an export with Board2D. An object x satisfying this concept may then be used as:

   Board2D board;
   board << CustomStyle( x.styleName(), x.defaultStyle() )
         << x;
