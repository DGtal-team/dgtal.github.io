Description of concept 'CImageContainer'

Aim: Defines the concept describing an image container.

Refinement of

Associated types :

  • Value: the type of values stored in the image.
  • Iterator: an iterator in the image.
  • ConstIterator: a const iterator in the image.
  • SpanIterator: a 1D span iterator in the image (not yet tested).

Notation

  • X : A type that is a model of CImageContainer
  • x, y : Object of type X
