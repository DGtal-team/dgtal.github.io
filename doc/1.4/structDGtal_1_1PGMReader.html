Detailed Description

template<typename TImageContainer, typename TFunctor = functors::Cast<typename TImageContainer::Value>>
struct DGtal::PGMReader< TImageContainer, TFunctor >

Aim: Import a 2D or 3D using the Netpbm formats (ASCII mode).

Description of class 'PGMReader'

  • PPM: RGB

    • PGM: grayscale
    • PPM3D: 3D variant of PPM
    • PGM3D: 3D variant of PGM

    Simple example: (extract from test file testPGMReader.cpp)

    #include "DGtal/helpers/StdDefs.h"
    #include "DGtal/io/readers/PGMReader.h"
    #include "DGtal/kernel/images/ImageSelector.h"
    ...
    string filename = "test.pgm";
    typedef ImageSelector < Z2i::Domain, uint>::Type Image;
    Image image = PGMReader<Image>::importPGM( filename );
