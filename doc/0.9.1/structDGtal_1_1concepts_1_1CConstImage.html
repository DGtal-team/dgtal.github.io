Detailed Description

template<typename I>
struct DGtal::concepts::CConstImage< I >

Aim: Defines the concept describing a read-only image, which is a refinement of CPointFunctor.

Description of concept 'CConstImage'

Refinement of CTrivialConstImage

Associated types :

  • Domain: type of the image domain, model of concept CDomain
  • ConstRange: type of range of image values, model of concept CConstBidirectionalRangeFromPoint

Notation
