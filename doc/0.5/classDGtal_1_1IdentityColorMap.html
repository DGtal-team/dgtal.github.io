Detailed Description

template<typename PValue>
class DGtal::IdentityColorMap< PValue >

Aim: This class template may be used to (linearly) convert scalar values in a given range into gray levels.

Description of template class 'IdentityColorMap'

The IdentityColorMap can be used either as a functor object (the value range is given at the object's construction) which converts a value into a Color structure, or it can be used through a static method taking both the range and the value as parameters.

The code below shows a possible use of this class. 

 #include "DGtal/io/Color.h"
 #include "IdentityColorMap.h"
 // ...
 {
   IdentityColorMap<float> grayShade(0.0f,1.0f);
   Color white = grayShade(1.0f);
   Color gray1 = grayShade(0.5f);
   // Or, equivalently:
   Color gray2 = IdentityColorMap<float>::getColor(0.0f,1.0f,0.5f);
 }
Template Parameters:
PValueThe type of the range values.
