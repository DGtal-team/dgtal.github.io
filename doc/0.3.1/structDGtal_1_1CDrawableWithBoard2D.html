Description of concept 'CDrawableWithBoard2D'

Aim: The concept CDrawableWithBoard2D specifies what are the classes that admit an export with Board2D. An object x satisfying this concept may then be used as:

   Board2D board;
   board << CustomStyle( x.styleName(), x.defaultStyle() )
         << x;

Notation

  • X : A type that is a model of CDrawableWithBoard2D
  • x, y : Object of type X
  • m : a string of characters
