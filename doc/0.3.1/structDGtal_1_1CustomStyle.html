Detailed Description

Modifier class in a Board2D stream. Useful to choose your own style for a given class. Realizes the concept CDrawableWithBoard2D.

Examples:

ctopo-1.cpp, ctopo-2.cpp, dgtalBoard2D-3-custom-classes.cpp, dgtalBoard2D-3-custom-points.cpp, dgtalBoard2D-4-colormaps.cpp, and freemanChainFromImage.cpp.
