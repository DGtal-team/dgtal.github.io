Detailed Description

Modifier class in a Board2D stream. Useful to choose your own mode for a given class. Realizes the concept CDrawableWithBoard2D.

Examples:

ctopo-1.cpp, dgtalBoard2D-2-sets.cpp, dgtalBoard2D-4-colormaps.cpp, freemanChainFromImage.cpp, and khalimskySpaceScanner.cpp.
