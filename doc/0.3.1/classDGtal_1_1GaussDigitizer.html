Detailed Description

template<typename TSpace, typename TEuclideanShape>
class DGtal::GaussDigitizer< TSpace, TEuclideanShape >

Aim: A class for computing the Gauss digitization of some Euclidean shape, i.e. its intersection with some . It is also a model of CPointPredicate and of CShape. Note that the real point (0,...,0) is mapped onto the digital point (0,...,0).
