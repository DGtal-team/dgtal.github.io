Detailed Description

template<typename T>
struct DGtal::CLocalEstimatorFromSurfelFunctor< T >

Aim: this concept describes functors on digtal surface surfel which can be used to define local estimator using the adapter LocalEstimatorFromSurfelFunctorAdapter.

Description of concept 'CLocalEstimatorFromSurfelFunctor'

Refinement of

Associated types :

  • Surfel: a type of Surfel
  • Embedder: a model of CSCellEmbedder
  • Quantity: a return type for the estimator (model of CQuantity)

Notation

  • X : A type that is a model of CLocalEstimatorFromSurfelFunctor
  • x, y : object of type X
  • aSurfel : object of type Surfel
