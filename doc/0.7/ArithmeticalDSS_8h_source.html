712  * \brief Aim: This class represents a standard digital straight segment (DSS),
713  * ie. the sequence of simply 4-connected digital points contained in a
714  * standard digital straight line (DSL) between two points of it.
715  * This class is an alias of ArithmeticalDSS.
716  *
717  * A 4-connected DSS can be declared and constructed as follows:
718  * @snippet geometry/curves/exampleArithmeticalDSS.cpp ArithmeticalDSSStandardCtor
719  *
720  * This requires the following include:
721  * @snippet geometry/curves/exampleArithmeticalDSS.cpp ArithmeticalDSSHeader
722  *
723  * See the documentation of NaiveDSS8 for a longer example
724  * and see @ref moduleArithDSSReco for further details.
725  *
726  * @tparam TCoordinate a model of integer for the DGtal point coordinates and
727  * the slope parameters.
728  * @tparam TInteger a model of integer for the intercepts and the remainders

730  *
