template<typename T>
struct DGtal::concepts::CWithGradientMap< T >

Aim: Such object provides a gradient map that associates to each argument some real vector.

Description of concept 'CWithGradientMap'

Refinement of

Associated types

  • Argument : the type of each element of the domain
  • RealVector : the type for representing the gradient values.
  • GradientMap : functor Argument -> RealVector, a model of CUnaryFunctor<T, Argument, RealVector > and boost::CopyConstructible.
  • gradientMap() const: returns a GradientMap.
