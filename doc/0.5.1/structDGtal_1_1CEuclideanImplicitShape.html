Detailed Description

template<typename T>
struct DGtal::CEuclideanImplicitShape< T >

Aim: characterizes models of digital implicit shapes. Implicit shapes are oriented shapes with an operator () which returns the implicit function value. Furthermore, negative (resp. positive or equal to zeeo) values of the implicit function correspond to DGtal::INSIDE value (resp DGtal::OUTSIDE or DGtal::ON)
